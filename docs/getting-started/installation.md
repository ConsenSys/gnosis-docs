# Npm

```
npm install gnosisjs
```

# Bower
```
bower install gnosisjs
```

# Build

```
$ cd gnosis.js
$ webpack
$ ls dist/gnosis.js
```

# Include javascript
```
<script src="gnosis.js/dist/gnosis.js"></script>
```

# Include nodejs
```
const gnosis = require('gnosisjs');
```


# Requirements

* NodeJS 5.0+ recommended.
* Windows, Linux or Mac OS X
